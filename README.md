# Vuga Platform

[![CircleCI](https://circleci.com/gh/Bahatisteven/vuga/tree/main.svg?style=svg)](https://circleci.com/gh/Bahatisteven/vuga/tree/main)
[![Coverage Status](https://coveralls.io/repos/github/Bahatisteven/vuga/badge.svg?branch=main&service=github)](https://coveralls.io/github/Bahatisteven/vuga?branch=main)

## Description

Vuga is a NestJS-based platform built with TypeScript and PostgreSQL.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## CI/CD

This project uses CircleCI for continuous integration and Coveralls for code coverage reporting.
